# CosmoSomething

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* 宇宙空間で戦闘機が敵を打つシューティングゲーム
* 

## ゲームの遊び方
* 矢印キーで戦闘機を操作し，スペースキーでビームを発射し敵を撃破する。
* 

## ゲームの実装
### 共通基本機能
* 背景画像とプレイヤー(戦闘機)の描画
